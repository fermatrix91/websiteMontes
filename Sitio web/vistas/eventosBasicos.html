Eventos JQuery

Sintaxis

Para utilizar JQuery necesitamos lo siguiente:
El símbolo utilizado para acceder a JQuery

$

El selector de nuestro elemento Html, escrito entre comillas

simples: 'selector'
dobles: "selector"

Y por último una acción, escrita luego de un punto. Es una buena práctica terminar la oración con el signo punto y coma ";"

.Accion();

$('selector').action();

Alertas

Son simples mensajes que se muestran por encima del contenido de nuestro sitio web, por lo general se utilizan para mostrar mensajes de advertencia a los visitantes del sitio. Se observa y confirma que el mensaje se muestra sobre el contenido Html debido a que llamos al mensaje dentro del evento Document Ready.
