¿Qué es JQuery?

Instalando JQuery

El primer paso es descargar nuestra biblioteca (un archivo con extensión .js), esto se hace desde la página oficial de Jquery. Demos clic al enlace de abajo

JQuery.js

Luego que estemos en la nueva pestaña podemos dar clic derecho y luego guardar como y con esto ya tenemos nuestra biblioteca JQuery.

Luego necesitamos hacer referencia a la biblioteca desde nuestro código Html de la siguiente manera.

El evento Document Ready

Este evento consiste en decirle al navegador que ejecute código JQuery hasta que haya terminado de cargarse todo el documento Html. Esto es esencial para evitar tener errores al momento de crear funciones en nuestro script y así el navegador cargará el contenido Html sin anomalías.

Alertas

Son simples mensajes que se muestran por encima del contenido de nuestro sitio web, por lo general se utilizan para mostrar mensajes de advertencia a los visitantes del sitio. Se observa y confirma que el mensaje se muestra sobre el contenido Html debido a que llamamos al mensaje dentro del evento Document Ready.
