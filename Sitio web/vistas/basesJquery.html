Bases para utilizar JQuery

Sintaxis

Para utilizar JQuery necesitamos lo siguiente:
El símbolo utilizado para acceder a JQuery

$

El selector de nuestro elemento Html, escrito entre comillas

simples: 'selector'
dobles: "selector"

Y por último una acción, escrita luego de un punto. Es una buena práctica terminar la oración con el signo punto y coma ";"

.Accion();

$('selector').action();

Selector por Id

Es el nombre de identificación ÚNICO de cada elemento html al que le asignemos.

El id del botón creado es botonEnviarFormulario. Es de suma importancia que cuando asignemos un "id" a un elemento Html, éste debe ser único en todo el documento para evitar resultados inesperados o incorrectos en el manejo de tales elementos.

Para leer un selector por "id" se hace así # (signo numeral) antes del nombre escogido como id

$("#botonEnviarFormulario")

Selector por Class

La clase es el nombre que podemos asignar a uno o más elementos a diferencia del id que es único. Si quisiéramos asignar un color de texto azul a más de un elemento podemos hacer uso de "class"

La clase que pueden tener todos los elementos a los que le asignemos es botonesAzules (puede ser cualquier nombre).

Para leer un selector por "class" se hace así . (punto) antes del nombre escogido como clase

$(".botonEnviarFormulario")

Selector por Elemento

El nombre del elemento Html es clave en este aspecto. Si quisiéramos hacer algo con todos los párrafos de nuestro documento Html, en lugar de colocar la misma clase a todos, podemos manipularlos con el nombre de su etiqueta, que por supuesto la comparten todos.
